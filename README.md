# cli
Command line tool for pushing code frames to api

To get started:

```bash
git clone https://github.com/iterableio/cli.git
cd cli
virtualenv env
. env/bin/activate.fish
make bootstrap
```

To test:

```bash
. env/bin/activate.fish
make test
```

To lint:

```bash
. env/bin/activate.fish
make lint
```

Good to go
